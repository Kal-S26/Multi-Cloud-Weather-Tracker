# 🌦️ Multi-Cloud Weather Tracker (AWS + Azure)

## 🚀 Overview
This project demonstrates how to deploy a weather tracking web application across **AWS and Azure** using Terraform (Infrastructure as Code).

## 🧰 Tech Stack
- AWS S3 (Static Website Hosting)
- Azure Blob Storage
- Terraform
- HTML, CSS, JavaScript

## 🌍 Features
- Multi-cloud deployment (AWS + Azure)
- Infrastructure automation using Terraform
- Static website hosting
- Scalable and cost-efficient design
